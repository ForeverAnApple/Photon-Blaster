# Photon Blaster!

## List of Components
* 1 x Photon
* 12v RGB LED strip (SMD5050)
* 3 x 10k Ohm Resistors
* 3 x Logic Level N-channel MOSFETs
* 1 x Breadboard
* Hookup wires
* 12v Power Supply - 8xAA Battery Pack
